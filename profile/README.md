## 🧭 What I Do

Most QA engineers test software. Most AI builders ship agents. I sit deliberately in the overlap: **I design quality processes, then build AI agents that run them at scale.**

| 🔍 Quality Engineering | 🤖 AI Agent Building | ⚡ Where They Meet |
|---|---|---|
| Test strategy & test-case design from raw business requirements | Custom agents & skills for **Claude Code** and **Microsoft Copilot** | AI agents that turn user stories into full test suites in minutes |
| Functional, regression, exploratory & UAT testing | Prompt architecture: system prompts, guardrails, structured outputs | Requirement-gap analysis agents that catch defects **before** code exists |
| Requirement review — finding gaps, ambiguity & missing acceptance criteria | Agent skill packs, MCP integrations & multi-agent workflows | Impact-analysis agents that predict what a change will break |
| Defect lifecycle management (JIRA / Azure DevOps) | Rapid prototyping with AI-first dev tools (Claude Code, Antigravity, Copilot) | Shift-left quality, automated by design |

## 🛠️ The "Ultra" Agent Suite — AI Agents I've Built for the SDLC

A family of production-grade agent skills that put a senior specialist inside the AI, covering the quality side of the software lifecycle end-to-end:

- **🧪 Ultra QA** — turns any user story, BRD, or JIRA/ADO ticket into structured, comprehensive test cases: functional, negative, boundary, and regression coverage with traceability back to requirements.
- **📋 Ultra Business Analyst** — reviews requirements like a principal BA: flags gaps, challenges ambiguity, validates acceptance criteria, and scores requirement completeness before a single line of code is written.
- **🏗️ Ultra Architect** — impact-analysis agent that evaluates new requirements against an existing product: what does this touch, what will it break, is it feasible.
- **✍️ Ultra Prompt Generator** — a prompt-engineering agent that designs production-ready system prompts for Copilot agents and other AI platforms.

> The thesis: quality isn't a phase, it's a pipeline — and every stage of it can be augmented by a well-designed agent.
